## 📚 What's Inside

PsychMind is a modern, interactive learning platform designed to help you master psychology principles through structured modules, interactive flashcards, and comprehensive practice quizzes.

### Features

- **9 Weekly Modules** — Complete coverage of psychology fundamentals:
  - Week 1: Introduction to Psychology (History, Perspectives & Research Methods)
  - Week 2: Biological Bases of Behavior (Neurons, Brain Structures & Genetics)
  - Week 3: Sensation & Perception (Senses, Thresholds & Perceptual Organization)
  - Week 4: States of Consciousness (Sleep, Dreams, Hypnosis & Drugs)
  - Week 5: Learning (Classical, Operant & Observational Learning)
  - Week 6: Memory (Encoding, Storage, Retrieval & Forgetting)
  - Week 7: Motivation & Emotion (Drives, Needs & Emotional Theories)
  - Week 9: Psychological Disorders (Anxiety, Mood, Personality & Psychotic Disorders)
  - Week 10: Treatment & Therapy (Psychotherapy, Biomedical & Lifestyle Interventions)

- **130+ Interactive Flashcards** — Flip cards for each concept with instant reveal and mastery tracking

- **60 Practice Quiz Questions** — Carefully curated questions covering all modules with:
  - Multiple choice format (A/B/C/D options)
  - Instant feedback with detailed explanations
  - Chapter-based filtering
  - Customizable quiz length (10, 20, 30, or 60 questions)
  - Score tracking and progress indicators

- **Modern Design System** — "Mindscape" design philosophy featuring:
  - Color-coded chapters for quick identification
  - Smooth animations and transitions
  - Responsive layout for all devices
  - Clean, academic-focused typography

- **Learning Path Timeline** — Visual journey from foundations to mastery

- **Progress Tracking** — Monitor your learning progress across all modules

## 🎨 Design Philosophy

The site follows the **"Mindscape"** design system:

- **Color Coding**: Each week has a unique color for visual organization
- **Typography**: Balanced blend of bold display fonts and readable body text
- **Spacing**: Generous whitespace for clarity and focus
- **Interactions**: Smooth transitions and micro-animations enhance engagement
- **Accessibility**: Keyboard navigation and clear visual hierarchy

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- pnpm 10+

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

The app will be available at `http://localhost:3000`

## 📖 How to Use

### Explore Chapters
1. Click "Chapters" in the navigation
2. Select any chapter to view its concepts
3. Click on concept cards to reveal definitions
4. Mark concepts as "mastered" to track progress

### Take Practice Quiz
1. Click "Practice Quiz" in the navigation
2. Select chapters to include (or select all)
3. Choose number of questions (10, 20, 30, or 60)
4. Click "Start Quiz"
5. Answer questions and get instant feedback
6. Review your score and explanations

### Track Progress
- Each chapter shows your mastery percentage
- Quiz results are displayed with detailed breakdowns
- Visual progress indicators throughout the site

## 📝 Content Coverage

All content is derived from comprehensive psychology course materials covering:

- **Research Methods**: Experimental design, bias, correlation vs. causation
- **Neuroscience**: Brain structure, neurotransmitters, nervous system
- **Sensation & Perception**: Sensory processes, perceptual organization, depth cues
- **Consciousness**: Sleep cycles, dreams, hypnosis, psychoactive drugs
- **Learning**: Classical conditioning, operant conditioning, observational learning
- **Memory**: Encoding, storage, retrieval, forgetting mechanisms
- **Motivation & Emotion**: Drive theories, emotion models, stress responses
- **Psychological Disorders**: Anxiety, mood, personality, and psychotic disorders
- **Treatment**: Psychotherapy approaches, biomedical treatments, lifestyle interventions
